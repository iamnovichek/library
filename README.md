
## Installation

1) Type in your terminal:

```bash
  pip3 install peewee
```

2) Type in your ```lib_manager.py``` :

    ```bash
    DB.create_tables(
        [
            Member,
            Book,
            CheckOutHistory,
            CheckedOutBook
        ]
    )
    ```


3) Execute your code

4) Delete :

    ```bash
    DB.create_tables(
        [
            Member,
            Book,
            CheckOutHistory,
            CheckedOutBook
        ]
    )
    ```

5) Congratulations! Now you can use "library manager" !
